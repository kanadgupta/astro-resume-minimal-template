---
title: Director of Automatic Programming
company: Remington Rand
period: Jun 1949 - Jul 1967
---

- Joined the Eckert–Mauchly Computer Corporation (later Remington Rand) as senior mathematician on the UNIVAC I, the first commercial electronic computer.
- Built the A-0 System, the first compiler, proving that a computer could translate human-readable symbolic code into machine instructions.
- Led development of successive automatic programming tools:
  - Designed FLOW-MATIC, the first English-like programming language, targeting business data processing.
  - Grew reusable subroutine libraries and compiler tooling that cut programming time from weeks to hours.
- Evangelized automatic programming to skeptical industry and government audiences, driving adoption of higher-level languages over hand-written machine code.
